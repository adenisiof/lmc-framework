[OBJETIVO]
Criar uma calculadora científica simples capaz de realizar operações matemáticas básicas e algumas funções científicas comuns.

[REQUISITOS]

* Deve conter uma área de exibição para mostrar os números digitados e o resultado.
* Deve conter botões numéricos de 0 a 9.
* Deve conter botões para as quatro operações básicas:
  adição, subtração, multiplicação e divisão.
* Deve conter botão para limpar a tela.
* Deve conter botão de igual para calcular o resultado.
* Deve impedir divisão por zero.
* Deve conter algumas funções científicas simples:
  raiz quadrada, potência ao quadrado, porcentagem e número pi.
* O código deve ser simples e fácil de entender.
* A interface deve ser organizada e adequada para navegador.
* A lógica deve ser feita em JavaScript puro, sem bibliotecas externas.

[FUNCOES_CIENTIFICAS]

* raiz_quadrada = calcular a raiz quadrada do número exibido
* potencia_quadrado = elevar o número exibido ao quadrado
* porcentagem = dividir o número exibido por 100
* pi = inserir o valor aproximado de PI

[SAIDA_ESPERADA]
Gerar três arquivos separados:

* index.html
* style.css
* script.js
